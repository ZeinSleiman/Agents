# Prompts

This folder contains prompt templates used by agents and workflows.

## Structure

Organize prompts by agent or use-case:

```
prompts/
├── README.md
└── my-use-case/
    ├── system.md        # System prompt
    └── user.md          # User prompt template
```

## Guidelines

- Use descriptive file names that reflect the prompt's purpose.
- Document any variables or placeholders at the top of each prompt file.
- Version-control prompts so that changes are traceable.
