# Agents

This repository contains all agents, skills, prompts, and agentic workflows used for AI-powered automation experiments.

## Repository Structure

| Folder | Purpose |
|--------|---------|
| [`agents/`](agents/) | Agent definitions and configurations |
| [`skills/`](skills/) | Reusable skills and tools that agents can invoke |
| [`prompts/`](prompts/) | Prompt templates used by agents and workflows |
| [`workflows/`](workflows/) | End-to-end agentic workflow definitions |

## Getting Started

1. Browse the folder for the type of artifact you are looking for (agent, skill, prompt, or workflow).
2. Each subfolder contains a `README.md` with details on structure and conventions.
3. See [`.github/copilot-instructions.md`](.github/copilot-instructions.md) for coding conventions and guidelines used throughout this repository.
